# Tutorials

This directory contains Diataxis-style tutorials: learning-oriented,
step-by-step exercises that help practitioners experience the KBPD workflow.

Use tutorials when you want to learn the flow by doing it. Use `.kbriefs/README.md`
for the knowledge-system reference, `AGENTS.md` for always-on agent rules, and
`.agents/skills/` for reusable agent workflows.

Start with:

- [Use KBPD Skills During Development](use-kbpd-skills-during-development.md)
